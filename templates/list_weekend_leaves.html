Import Învoiri Weekend în Masă

Introduceți datele învoirilor, fiecare pe o singură linie, în formatul:

Grad Nume Prenume DD.MM.YYYY HH:MM-HH:MM, DD.MM.YYYY HH:MM-HH:MM, ..., biserica (opțional)

Exemple:

M.m.IV Renț Francisc 04.07.2025 15:00-20:00, 06.07.2025 09:00-11:00, biserica
Sdt. Popescu Vasile 04.07.2025 16:00-22:00, 05.07.2025 10:00-22:00

Până la 3 intervale de date per student, care trebuie să corespundă unui weekend logic (Vineri, Sâmbătă, Duminică). Cuvântul "biserica" la final este opțional și se aplică dacă Duminica este una din zilele selectate.

Format date: DD.MM.YYYY. Format ore: HH:MM.
